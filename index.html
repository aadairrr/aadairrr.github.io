Stat Together

Stat Together is an iOS app created to help better track in-game volleyball statistics.
Stat Together is currently used by hundreds of teams across the country. There are no other available apps that can handle tacking stats on multiple mobile devices at once. Learn more about Stat Together here.


Stat Together's frontend was written in Swift 5 and backend functions are handled written with TypeScript using Google Cloud. Multiple devices can connect in real time to the database allowing faster stat-tracking and real-time game analytics.

Color Together

A small React App integrated with a real-time database backend. Multiple people can connect at once and "paint" on a shared grid of pixels. Inspired by Reddit's "Reddit Place" and created to learn more about React and handling asynch updates and database reads/writes.
